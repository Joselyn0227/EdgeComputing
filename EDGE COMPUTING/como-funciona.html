¿Cómo Funciona?

El Edge Computing utiliza una red distribuida de dispositivos y servidores locales que procesan datos cerca del punto de origen. Estos nodos de borde actúan como intermediarios entre los dispositivos IoT y la nube central.

Los datos se procesan en tres niveles: dispositivo edge (sensores y actuadores), nodos edge (servidores locales) y nube central. Cada nivel maneja diferentes tipos de procesamiento según la urgencia y complejidad de los datos.

Los datos se recopilan en dispositivos edge, se procesan inicialmente en nodos locales para decisiones inmediatas, y solo la información relevante se envía a la nube para análisis más profundos y almacenamiento a largo plazo.
